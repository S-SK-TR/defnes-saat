# Defne's Saat

Saat okuma becerilerini geliştirmek için interaktif PWA uygulaması

## 🚀 Özellikler

- 🕒 Gerçek zamanlı analog ve dijital saat gösterimi
- 🎲 Rastgele saat belirleme
- 📊 Saat okuma pratik modu
- 📱 Mobil uyumlu responsive tasarım
- 🌐 Offline kullanım desteği
- 🖥️ Tam ekran modu

## 🛠️ Kurulum

1. Bu repo'yu klonlayın:
```bash
git clone https://github.com/[kullanici-adi]/defnes-saat.git
```

2. Gerekli bağımlılıkları yükleyin:
```bash
npm install
```

3. Uygulamayı başlatın:
```bash
npm start
```

## 📦 Vercel Deploy

Uygulama otomatik olarak Vercel'e deploy edilecektir. Deploy tamamlandığında link paylaşılacaktır.
